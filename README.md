# E-Commerce Backend Service
This repository contains the REST API service that is used for a e-commerce platform. Built with Java / Spring Boot, it handles the core business logic required for the storefront to function. The API manages the product catalog, processes checkout workflows, and securely handles stock quantity updates to ensure inventory remains accurate during user transactions.

Note: This API is designed to integrate directly with a React/Vite frontend application which I made.

### Tech Stack

* **Core Framework:** Java 25 / Spring Boot
* **API:** Spring Web MVC
* **Database:** PostgreSQL
* **ORM:** Spring Data JPA
* **Build Tool:** Maven
* **Utilities:** Lombok
* **Testing:** Spring Boot Test

## Build & Run Instructions
To boot the application locally:  
mvn spring-boot:run

### The Postman Collection
A complete Postman collection is included in the root of this repository: SpringEcom.postman_collection.json.
